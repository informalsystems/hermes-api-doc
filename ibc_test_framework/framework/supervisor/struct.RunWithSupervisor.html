pub struct RunWithSupervisor<'a, Test> {
    pub test: &'a Test,
}
Expand description

A wrapper type that implements the same test traits as the wrapped Test type, and spawns the relayer supervisor before the inner test is called.

For example, if Test implements BinaryChannelTest, then RunWithSupervisor<Test> also implements BinaryChannelTest.

The automatic spawning of supervisor can be disabled by implementing SupervisorOverride and returning false.

When composing the test runners with RunWithSupervisor, it is important to ensure that RunWithSupervisor do not appear more than once in the nesting. Otherwise the supervisor may spawn more than once during tests.
