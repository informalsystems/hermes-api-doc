Bootstraps two relayer chain handles with connected foreign clients.

Returns a tuple consisting of the RelayerDriver and a ConnectedChains object that contains the given full nodes together with the corresponding two ChainHandles and ForeignClients.

This method gives the caller a way to modify the relayer configuration that is pre-generated from the configurations of the full nodes.
