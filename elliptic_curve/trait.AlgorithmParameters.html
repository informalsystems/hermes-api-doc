Associate an ObjectIdentifier (OID) with an elliptic curve algorithm implementation.

This is used as as the parameters of an AlgorithmIdentifier as described in RFC 5280 Section 4.1.1.2: https://tools.ietf.org/html/rfc5280#section-4.1.1.2
