  • requires_legacy_upgrade_proposal
    Looks at the ibc-go version to determine if the legacy UpgradeProposal message or if the newer MsgIBCSoftwareUpdate message should be used to upgrade the chain. If the ibc-go version returned isn’t reliable, a deprecated version, then the version of Cosmos SDK is used, if any. If there is no SDK version, we assume that the legacy upgrade is required.
