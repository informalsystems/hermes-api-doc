  • ibc_event_try_from_abci_event
    Note: This function, as well as other helpers, are needed as a workaround to Rust’s orphan rule. That is, we want the AbciEvent -> IbcEvent to be defined in the relayer crate, but can’t because neither AbciEvent nor IbcEvent are defined in this crate. Hence, we are forced to make an ad-hoc function for it.
