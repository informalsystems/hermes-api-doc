Analyze the request’s headers and body.

This module provides functions and sub-modules that allow you to easily analyze or parse the request’s headers and body.

  • In order to parse JSON, see the json module.
  • In order to parse input from HTML forms, see the post module.
  • In order to read a plain text body, see the plain_text_body function.
